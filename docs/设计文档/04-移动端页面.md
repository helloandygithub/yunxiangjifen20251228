# 云享积分平台 - 设计提示词（4/4）

## 📱 移动端H5首页

```
设计移动端（H5/小程序）积分平台首页。
宽度：375px（iPhone标准）

页面布局（从上到下）：

1. 顶部用户栏（橙色渐变背景）
   - 左侧：头像 + "下午好"问候语
   - 右侧：当前积分展示

2. 快捷功能区（白色卡片）
   - 4个圆形图标按钮（2x2网格）
   - 积分兑换、积分明细、我的提交、我的订单

3. 热门活动区
   - 标题"热门活动" + "更多"链接
   - 垂直列表卡片
   - 每个卡片：左侧缩略图 + 右侧标题描述积分

4. 底部TabBar
   - 4个标签：首页、活动、商城、我的
   - 选中状态橙色图标+文字

设计要点：
- 适配移动端触摸操作
- 按钮点击区域44px以上
- 卡片间距16px
- 整体温暖友好，橙黄色主题
```

## 📱 移动端活动列表页

```
设计移动端活动列表页面。

页面布局：

1. 顶部搜索栏
   - 搜索输入框 + 筛选图标

2. 分类Tab
   - 横向滚动标签：全部、云服务器、云数据库、SaaS、信创

3. 活动卡片列表（垂直排列）
   每个卡片：
   - 封面图（16:9比例）
   - 状态标签（进行中）
   - 标题 + 描述
   - 底部：积分奖励 + 参与按钮

设计要点：
- 下拉刷新、上拉加载更多
- 卡片圆角12px
- 间距适中便于点击
```

## 📱 移动端积分商城页

```
设计移动端积分商城页面。

页面布局：

1. 顶部积分卡片（橙色渐变）

```
