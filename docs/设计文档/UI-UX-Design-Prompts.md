## 📌 整体设计基调

```
品牌名称：云享积分
业务背景：企业数字化产品推广平台，推广云服务器、云数据库、IaaS、PaaS、SaaS、信创产品等
目标用户：企业老板、采购经理、IT负责人、人事、财务、行政等企业决策者和执行者
设计调性：专业可信赖、科技感、简洁高效、企业级品质

配色方案：
- 主色：橙黄色 #F5A623（温暖、活力、积分主题）
- 辅色：科技蓝 #4A90E2（专业、科技、云服务）
- 背景：米白色 #FFFBF5 / 浅灰 #F8F9FA
- 文字：深灰 #333333、中灰 #666666

字体：思源黑体/苹方，中文友好，专业商务感
```

## 🖥️ PC端页面

### 1. PC端首页 (Home Page)

```
设计一个企业数字化产品积分推广平台的PC端首页。

品牌：云享积分
用户群体：企业老板、采购、IT负责人等企业用户
业务：通过参与云服务产品（云服务器、云数据库、SaaS等）推广活动赚取积分，兑换礼品

页面布局（从上到下）：

1. 顶部导航栏
   - 左侧：橙色圆形Logo + "云享积分"品牌名
   - 中间：导航菜单（首页、活动专区、积分商城、我的积分）带橙色高亮背景
   - 右侧：积分显示（橙色背景圆角标签）+ 用户头像下拉菜单 + 登录按钮

2. Hero主视觉区域（米白色渐变背景）
   - 左侧60%：
     - 小字欢迎语"欢迎回来，数字化先锋"
     - 大标题"推广数字化产品 赚取丰厚积分"（48px加粗）
     - 副标题"参与云服务器、云数据库、SaaS等产品推广活动，轻松获得积分奖励"
     - 两个数据统计卡片并排（已完成任务数、累计获得积分）
     - 两个CTA按钮：橙色实心"前往活动专区" + 橙色边框"浏览积分商城"
   - 右侧40%：
     - 白色圆角卡片，展示"今日可获积分"数字（大字号橙色）
     - 下方小卡片展示"热门活动"快捷入口

3. 快速导航区域（白色背景）
   - 标题"快速导航" + 副标题"一键进入核心功能区域"
   - 3个并排卡片：
     - 活动专区（奖杯图标）- 参与推广赚积分
     - 积分商城（礼品图标）- 兑换数码好礼
     - 我的积分（用户图标）- 查看积分明细

4. 热门活动区域（浅米色背景）
   - 标题"热门活动" + 副标题"精选推广任务，快来参与"
   - 右上角"查看全部活动"链接
   - 3列活动卡片网格：
     - 卡片封面图（展示云服务、数据库、会议等产品图）
     - "进行中"状态标签
     - 活动标题（如"腾讯云服务器推广"、"企业微信推荐"）
     - 描述文字
     - 底部：积分奖励（橙色）+ 参与人数
     - 橙色"立即参与"按钮

5. 底部Footer
   - 品牌信息、快速链接、帮助支持、关注我们
   - 版权信息、ICP备案号

设计风格：
- 企业级专业感，简洁大气
- 橙黄色主题传递积分活力
- 圆角卡片（12-16px）
- 适度留白，信息层级清晰
- 图标使用线性风格
- 适合1920px宽屏展示
```

### 2. 活动列表页 (Activities List)

```
设计企业数字化产品推广活动列表页面。

页面结构：

1. 页面标题区
   - 大标题"活动专区"
   - 副标题"参与企业数字化产品推广，轻松赚取积分"

2. 筛选栏
   - 标签筛选：全部、云服务器、云数据库、SaaS应用、信创产品、企业服务
   - 状态筛选：全部、进行中、即将开始、已结束
   - 排序：最新、积分最高、参与最多

3. 活动卡片网格（3列）
   每个卡片包含：
   - 封面图片（展示对应产品：服务器机房、数据库图标、SaaS界面等）
   - 右上角状态标签（进行中/即将开始）
   - 活动标题（如"腾讯云CVM推广计划"、"企业微信拉新活动"）
   - 活动描述（2行截断）
   - 产品分类标签（云服务器/SaaS等）
   - 底部信息栏：
     - 左侧：积分奖励（橙色）"+500积分"
     - 右侧：已参与人数
   - 橙色"立即参与"按钮

4. 分页器
   - 居中显示，橙色主题

设计要点：
- 活动封面图要体现企业科技感
- 卡片hover效果：轻微上浮 + 阴影加深
- 空状态展示"暂无活动"友好提示
- 筛选标签选中状态用橙色背景
```

### 3. 活动详情页 (Activity Detail)

```
设计企业数字化产品推广活动详情页面。

页面布局：

1. 面包屑导航
   首页 > 活动专区 > 腾讯云服务器推广计划

2. 活动信息区（左右布局）
   - 左侧40%：活动封面大图（展示产品，如云服务器界面）
   - 右侧60%：
     - 活动标题"腾讯云服务器推广计划"（大字号）
     - 产品标签"云服务器"（橙色背景）
     - 活动描述（详细说明推广规则）
     - 信息卡片：
       - 积分奖励：+500积分/单
       - 活动时间：2025.01.01 - 2025.03.31
       - 参与次数：每日3次
       - 已参与：1,250人

3. 任务表单区（白色卡片）
   - 标题"填写推广信息"
   - 表单字段（根据活动类型动态展示）：
     - 客户公司名称（文本输入）
     - 客户联系人（文本输入）
     - 联系电话（手机号输入）
     - 推广产品（下拉选择）
     - 订单截图（图片上传区域，支持拖拽）
     - 订单金额（数字输入）
     - 备注说明（多行文本）
   - 提交按钮（橙色，宽度100%）

4. 活动规则区
   - 折叠面板展示活动规则、积分发放说明、注意事项

设计要点：
- 表单设计要简洁，标签清晰
- 上传区域用虚线边框+云朵图标
- 必填字段用红色星号标注
- 表单验证错误用红色提示
- 整体布局适合填写操作
```

### 4. 积分商城页 (Mall)

```
设计企业用户积分兑换商城页面。

商品类型：数码产品、办公用品、企业服务、虚拟权益等

页面布局：

1. 商城头部
   - 用户积分展示卡片（橙色渐变背景）
     - "我的积分"标题
     - 大号积分数字
     - "查看积分明细"链接

2. 分类导航
   - 标签式切换：全部、数码产品、办公用品、企业服务、虚拟权益
   - 选中状态橙色背景

3. 商品网格（4列）
   每个商品卡片：
   - 商品图片（正方形，圆角）
   - 库存不足时显示"库存紧张"标签
   - 商品名称（如"华为蓝牙耳机"、"京东E卡100元"、"腾讯会议月卡"）
   - 兑换所需积分（橙色大字号）
   - 库存数量（灰色小字）
   - 悬浮显示"立即兑换"按钮

4. 分页器

商品建议（符合企业用户）：
- 数码：蓝牙耳机、移动电源、U盘、键鼠套装
- 办公：笔记本、签字笔套装、桌面收纳
- 企业服务：腾讯会议月卡、企业网盘空间、在线文档会员
- 虚拟权益：京东E卡、星巴克券、视频会员

设计要点：
- 积分数字用橙色突出显示
- 商品图片统一比例和风格
- 已售罄商品灰度显示+售罄标签
- 整体氛围专业不失活力
```

### 6. 个人中心页 (User Center)

```
设计企业用户个人中心页面。

页面布局（左右分栏）：

1. 左侧用户卡片（30%）
   - 用户头像（圆形）
   - 手机号（中间四位隐藏）
   - 企业名称（如已填写）
   
   - 积分卡片（橙色渐变）
     - "当前积分"
     - 大号积分数字
     - "查看明细"按钮
   
   - 邀请卡片
     - "我的邀请码"
     - 邀请码（大字号）
     - "复制邀请码"按钮
     - 已邀请人数统计

2. 右侧Tab内容区（70%）
   Tab切换：积分明细 | 我的订单 | 活动记录
   
   积分明细Tab：
   - 表格展示：时间、类型、说明、积分变动（+/-颜色区分）
   
   我的订单Tab：
   - 订单卡片列表
   - 订单号、商品图片、商品名称、消耗积分
   - 状态标签（待发货/已发货/已完成）
   - 物流信息（如有）
   
   活动记录Tab：
   - 活动卡片列表
   - 活动名称、提交时间
   - 状态标签（审核中/已通过/未通过）
   - 获得积分（已通过时显示）

设计要点：
- 积分变动正数绿色、负数红色
- 状态标签用不同颜色区分
- 表格/列表简洁易读
- 支持按时间筛选
```
