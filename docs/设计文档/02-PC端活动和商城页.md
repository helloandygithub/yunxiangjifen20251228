# 云享积分平台 - 设计提示词（2/4）

## 🖥️ 活动详情页 (Activity Detail)

```
设计企业数字化产品推广活动详情页面。

页面布局：

1. 面包屑导航
   首页 > 活动专区 > 腾讯云服务器推广计划

2. 活动信息区（左右布局）
   - 左侧40%：活动封面大图（展示产品，如云服务器界面）
   - 右侧60%：
     - 活动标题"腾讯云服务器推广计划"（大字号）
     - 产品标签"云服务器"（橙色背景）
     - 活动描述（详细说明推广规则）
     - 信息卡片：
       - 积分奖励：+500积分/单
       - 活动时间：2025.01.01 - 2025.03.31
       - 参与次数：每日3次
       - 已参与：1,250人

3. 任务表单区（白色卡片）
   - 标题"填写推广信息"
   - 表单字段（根据活动类型动态展示）：
     - 客户公司名称（文本输入）
     - 客户联系人（文本输入）
     - 联系电话（手机号输入）
     - 推广产品（下拉选择）
     - 订单截图（图片上传区域，支持拖拽）
     - 订单金额（数字输入）
     - 备注说明（多行文本）
   - 提交按钮（橙色，宽度100%）

4. 活动规则区
   - 折叠面板展示活动规则、积分发放说明、注意事项

设计要点：
- 表单设计要简洁，标签清晰
- 上传区域用虚线边框+云朵图标
- 必填字段用红色星号标注
- 表单验证错误用红色提示
- 整体布局适合填写操作
```

## 🖥️ 积分商城页 (Mall)

```
设计企业用户积分兑换商城页面。

商品类型：数码产品、办公用品、企业服务、虚拟权益等

页面布局：

1. 商城头部
   - 用户积分展示卡片（橙色渐变背景）
     - "我的积分"标题
     - 大号积分数字
     - "查看积分明细"链接

2. 分类导航
   - 标签式切换：全部、数码产品、办公用品、企业服务、虚拟权益
   - 选中状态橙色背景

3. 商品网格（4列）
   每个商品卡片：
   - 商品图片（正方形，圆角）
   - 库存不足时显示"库存紧张"标签
   - 商品名称（如"华为蓝牙耳机"、"京东E卡100元"、"腾讯会议月卡"）
   - 兑换所需积分（橙色大字号）
   - 库存数量（灰色小字）
   - 悬浮显示"立即兑换"按钮

4. 分页器

商品建议（符合企业用户）：
- 数码：蓝牙耳机、移动电源、U盘、键鼠套装
- 办公：笔记本、签字笔套装、桌面收纳
- 企业服务：腾讯会议月卡、企业网盘空间、在线文档会员
- 虚拟权益：京东E卡、星巴克券、视频会员

设计要点：
- 积分数字用橙色突出显示
- 商品图片统一比例和风格
- 已售罄商品灰度显示+售罄标签
- 整体氛围专业不失活力
```

## 🖥️ 商品详情页 (Product Detail)

```
设计积分商城商品详情页面。

页面布局（左右分栏）：

1. 左侧商品展示（45%）
   - 大图展示区（正方形圆角）
   - 缩略图列表（如有多图）

2. 右侧商品信息（55%）
   - 商品名称（大标题）
   - 商品类型标签（实物/虚拟）
   - 兑换积分（橙色超大字号）
   - 库存信息
   - 商品描述
   
   - 分割线
   
   - 如果是实物商品，显示收货信息表单：
     - 收货人姓名
     - 联系电话
     - 收货地址（省市区+详细地址）
   
   - 橙色"立即兑换"大按钮
   - 按钮下方提示：兑换后积分将立即扣除

3. 商品详情Tab
   - 商品详情（图文介绍）
   - 兑换说明
   - 常见问题

设计要点：
- 积分数字要醒目
- 库存紧张时显示红色提示
- 积分不足时按钮置灰+提示"积分不足"
- 表单简洁易填写
```
