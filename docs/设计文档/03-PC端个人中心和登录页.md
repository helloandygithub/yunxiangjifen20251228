# 云享积分平台 - 设计提示词（3/4）

## 🖥️ 个人中心页 (User Center)

```
设计企业用户个人中心页面。

页面布局（左右分栏）：

1. 左侧用户卡片（30%）
   - 用户头像（圆形）
   - 手机号（中间四位隐藏）
   - 企业名称（如已填写）
   
   - 积分卡片（橙色渐变）
     - "当前积分"
     - 大号积分数字
     - "查看明细"按钮
   
   - 邀请卡片
     - "我的邀请码"
     - 邀请码（大字号）
     - "复制邀请码"按钮
     - 已邀请人数统计

2. 右侧Tab内容区（70%）
   Tab切换：积分明细 | 我的订单 | 活动记录
   
   积分明细Tab：
   - 表格展示：时间、类型、说明、积分变动（+/-颜色区分）
   
   我的订单Tab：
   - 订单卡片列表
   - 订单号、商品图片、商品名称、消耗积分
   - 状态标签（待发货/已发货/已完成）
   - 物流信息（如有）
   
   活动记录Tab：
   - 活动卡片列表
   - 活动名称、提交时间
   - 状态标签（审核中/已通过/未通过）
   - 获得积分（已通过时显示）

设计要点：
- 积分变动正数绿色、负数红色
- 状态标签用不同颜色区分
- 表格/列表简洁易读
- 支持按时间筛选
```

## 🖥️ 登录页 (Login)

```
设计企业级积分平台登录页面。

页面布局（居中卡片式）：
- 背景：橙色渐变（左上到右下）或科技感图案
- 登录卡片（白色圆角，居中）
  - 品牌Logo + "云享积分"
  - 副标题"企业数字化推广积分平台"
  
  - 登录表单：
    - 手机号输入框（带手机图标前缀）
    - 验证码输入框 + "获取验证码"按钮
    - 邀请码输入框（选填，带标签图标）
  
  - 橙色"登录/注册"大按钮
  
  - 底部协议勾选
    - "登录即表示同意《用户协议》和《隐私政策》"

设计要点：
- 登录卡片宽度400px左右
- 输入框高度适中，便于点击
- 获取验证码按钮倒计时状态
- 整体简洁专业，体现企业级品质
- 背景可加入淡淡的云/数据流元素
```
